# PatternPath (v2)

PatternPath is a structured platform for mastering Data Structures and Algorithms patterns.

## Version 2 Features
- **Refined UI**: Luxury dark mode design with gold/red accents.
- **Dynamic Templates**: Pattern pages generated from a master `arrays.html` template.
- **Improved Structure**: Organized codebase in `version 2/` directory.

## Setup
1. Open `version 2/index.html` in your browser.
2. Navigate to Dashboard > Patterns to see the generated content.
3. Use `scripts/generate_patterns.py` to regenerate pages if needed.

## Tech Stack
- HTML5, Tailwind CSS
- Vanilla JavaScript
- Python (for static generation)
